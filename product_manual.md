# 出海故事 - AI品牌基因引擎产品文档

## 1. 项目简介
“出海故事”是一个专为中小出海商家打造的AI品牌基因引擎。通过深度挖掘企业与产品的核心价值，该引擎能够一键生成体系化的品牌定位、视觉风格、品牌故事与营销内容。本原型演示了从用户输入到品牌全案生成的完整前端交互流程。

## 2. 交互流程
整个产品体验分为三个主要阶段：

1.  **品牌主页 (Landing Page)**
    *   展示产品核心价值主张（“让品牌构建简单如呼吸”）。
    *   介绍三大核心功能：品牌基因挖掘、视觉体系生成、营销全案落地。
    *   引导用户点击“开始生成”进入输入流程。

2.  **信息采集页 (Input Page)**
    *   用户填写企业基本信息：
        *   **企业名称**：品牌的唯一标识。
        *   **企业文字介绍**：包含背景、业务、初心等，作为AI分析的文本依据。
        *   **核心产品图片**：上传产品图，作为AI分析视觉风格的依据。
    *   提交后，系统模拟AI分析过程（Loading动画），随后跳转至生成结果页。

3.  **品牌全案结果页 (Result Page)**
    *   展示生成的品牌策略视觉板。
    *   **动态生成内容**：
        *   页面标题与Logo替换为用户输入的**企业名称**。
        *   品牌故事、起源故事等文本模块根据用户的**企业介绍**动态生成。
        *   “生成图片素材”功能中展示用户上传的**产品图片**。
    *   **静态模板内容**（演示用）：
        *   保留了高质感的木质纹理设计风格作为默认模板。
        *   展示了标准的色彩系统、字体系统和Slogan示例。

## 3. 页面结构与文件说明

*   **index.html**: 品牌故事主页。响应式设计，包含Hero区域、功能特性网格和页脚。
*   **input.html**: 信息采集页面。包含表单验证、图片本地预览（FileReader）和加载状态交互。
*   **result.html**: 生成结果展示页。基于`brand4_木质沙发.html`改造，增加了动态数据渲染逻辑。
*   **brand4_木质沙发.html**: 原始设计稿参考（保留）。

## 4. 技术实现说明

本项目采用纯前端技术栈实现，无需后端服务即可演示完整流程：

*   **数据持久化**：使用浏览器 `localStorage` 在页面间传递用户输入的数据（企业名、介绍、图片Base64）。
*   **动态渲染**：`result.html` 加载时读取本地存储的数据，并通过 DOM 操作替换页面中的关键文本和图片资源。
*   **样式设计**：使用 CSS 变量定义配色系统，确保视觉风格的一致性；采用 Flexbox 和 Grid 布局实现响应式适配。

## 5. 如何运行

1.  确保项目文件夹中包含所有 HTML 文件及引用的静态资源（图片/视频）。
2.  双击打开 `index.html` 即可开始体验。
3.  按照流程点击“开始生成” -> 填写表单 -> 查看结果。
