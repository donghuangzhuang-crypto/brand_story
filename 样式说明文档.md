# 极简卡片式报告设计样式规范

## 概述

本文档基于 `minimal_report_style_guide.html` 总结了一套极简、浅色系、干净高级的系统样式规范，专门用于企业网红出海报告的设计指导。

## 设计理念

- **极简主义**：去除冗余元素，专注核心信息展示
- **浅色系**：以白色和浅灰色为主，营造干净专业的视觉效果
- **卡片化布局**：模块化设计，信息层次清晰
- **现代感**：圆角、阴影、间距的精心设计

## icon 图标
-不准用表情包，尤其是带颜色的
-而是用最简单的图标，比如字体图标或是黑白的极简 icon


## 设计令牌 (Design Tokens)

### 颜色系统

#### 浅色主题
- **背景色**: `#fafafa` - 极浅灰，提供舒适的阅读环境
- **表面色**: `#ffffff` - 纯白，卡片和组件背景
- **主文本**: `#111111` - 近黑色，确保良好对比度
- **次要文本**: `#6b7280` - 中性灰，用于辅助信息
- **分割线**: `#e5e7eb` - 浅灰线条，subtle分隔
- **芯片背景**: `#f3f4f6` - 极浅灰，标签和按钮背景
- **强调色**: `#111111` - 与主文本同色，保持一致性
- **成功色**: `#0ea5e9` - 天蓝色，积极状态指示
- **警告色**: `#f59e0b` - 琥珀色，注意状态指示

#### 深色主题（可选）
- **背景色**: `#0a0a0a`
- **表面色**: `#111111`
- **主文本**: `#f5f5f5`
- **次要文本**: `#9ca3af`

### 几何属性

- **圆角半径**: `16px` - 统一的圆角，现代感
- **阴影**: `0 1px 1px rgba(17,17,17,.04), 0 6px 24px rgba(17,17,17,.06)` - 轻微立体感
- **字体栈**: `ui-sans-serif, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`

## 布局系统

### 网格布局

```css
.container {
  display: grid;
  grid-template-columns: 260px 1fr;
  gap: 24px;
  max-width: 1200px;
  margin: 0 auto;
  padding: 24px;
}
```

- **侧边栏宽度**: 260px
- **主内容区**: 自适应剩余空间
- **间距**: 24px 统一间距
- **最大宽度**: 1200px，居中对齐

### 响应式设计

- **断点**: 960px
- **小屏幕**: 单列布局，KPI网格调整为2列

## 组件规范

### 卡片 (Card)

```css
.card {
  background: var(--surface);
  border: 1px solid var(--line);
  border-radius: var(--radius);
  box-shadow: var(--shadow);
  padding: 16px;
}
```

**特点**:
- 白色背景
- 1px 浅灰边框
- 16px 圆角
- 轻微阴影
- 16px 内边距
- 卡片间 16px 间距

### 侧边栏步骤 (Sidebar Steps)

**设计要点**:
- 粘性定位，跟随滚动
- 圆点状态指示器（10px圆形）
- 三种状态：完成（成功色）、进行中（芯片背景）、待办（弱强调色）
- 14px 标题字体，12px 描述字体

### 顶部栏 (Header)

**布局**:
- 左侧：品牌名 + 面包屑导航
- 右侧：工具按钮组
- 32px 方形图标按钮，10px 圆角

### 芯片标签 (Chips)

```css
.chip {
  font-size: 12px;
  padding: 6px 10px;
  border-radius: 999px;
  background: var(--chip);
  border: 1px solid var(--line);
}
```

**特点**:
- 胶囊形状（999px 圆角）
- 12px 字体
- 6px 垂直，10px 水平内边距

### KPI 指标网格

```css
.kpis {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 12px;
}
```

**设计**:
- 4列等宽网格
- 22px 粗体数值
- 12px 灰色标签
- 12px 圆角边框

### 时间线 Feed

**特点**:
- 22px 圆形图标容器
- 虚线分割线
- 左侧图标 + 右侧内容布局
- 600字重标题，13px 灰色描述

### 表格

**样式**:
- 无外边框设计
- 仅底部分割线
- 10px 单元格内边距
- 12px 灰色表头
- 14px 表体字体

### 按钮系统

#### 默认按钮
```css
.btn {
  padding: 10px 14px;
  border-radius: 12px;
  border: 1px solid var(--line);
  background: var(--surface);
  font-weight: 600;
}
```

#### 主要按钮
```css
.btn.primary {
  background: var(--text);
  color: var(--bg);
  border-color: var(--text);
}
```

### CTA 区域

**设计**:
- 芯片背景色
- 左侧文案 + 右侧按钮组
- 14px 内边距
- 12px 灰色说明文字

## 交互状态

### 悬停效果
- 按钮：轻微阴影增强
- 链接：颜色变化
- 卡片：可选择性阴影加深

### 激活状态
- 步骤项：芯片背景高亮
- 按钮：轻微按下效果

## 应用指导

### 适用场景
- 企业数据报告
- 分析仪表板
- 项目进度展示
- 商业洞察报告
- 网红营销分析

### 设计原则
1. **信息层次**: 使用字重、颜色、尺寸建立清晰层次
2. **留白运用**: 充分利用空白空间，避免拥挤
3. **一致性**: 统一的圆角、间距、颜色使用
4. **可读性**: 确保足够的对比度和合适的字体大小
5. **响应式**: 考虑不同屏幕尺寸的适配

### 扩展建议
- 可根据品牌色调整强调色
- 保持整体浅色调和极简风格
- 新增组件时遵循现有设计令牌
- 考虑无障碍访问需求

## 技术实现

### CSS 变量系统
使用 CSS 自定义属性 (`:root`) 管理设计令牌，便于主题切换和维护。

### 响应式策略
采用 CSS Grid 和 Flexbox 结合，确保在不同设备上的良好表现。

### 性能考虑
- 最小化 CSS 体积
- 使用系统字体栈
- 优化阴影和动画性能
